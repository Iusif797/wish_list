# Продуктовые решения

**Пустой вишлист:** Плейсхолдер с иллюстрацией и CTA «Добавить первый подарок» — пользователь сразу понимает, что делать дальше.

**Гость по ссылке:** Просмотр без регистрации; резервирование и вклад — по анонимному токену в localStorage. Нет лишнего friction для друзей.

**Товар удалён с сайта:** Резервы и вклады сохраняются. При ошибке загрузки картинки показывается плейсхолдер. Владелец может отредактировать или удалить товар.
